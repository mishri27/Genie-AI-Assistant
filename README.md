# GenieAI Chatbot

GenieAI is a modern chatbot built using HTML, CSS, JavaScript, and the Google Gemini API.

## Features

* Modern responsive UI
* AI-powered conversations
* Enter key support
* Real-time responses
* Error handling
* Clean and professional design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Google Gemini API

## Installation

1. Clone the repository
2. Add your Gemini API key
3. Open the project using Live Server

## Future Improvements

* Chat history
* Dark/Light mode
* Voice input
* Multiple AI models
* User authentication
